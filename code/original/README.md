# Original KG²RAG Code

This directory contains the **original KG²RAG implementation** from the NAACL 2025 paper:

**Zhu, X., Xie, Y., Liu, Y., Li, Y., & Hu, W. (2025). Knowledge Graph-Guided Retrieval Augmented Generation. NAACL.**

## Files

- `kg_rag_distractor.py`: Original KG²RAG pipeline for distractor setting
- `kg_rag_full.py`: Original KG²RAG pipeline for fullwiki setting

## Usage

These scripts are the baseline implementations. For experimental extensions, see the `experimental/` directory.
